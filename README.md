# README


1. Create config.js in your javascripts folder
2. Go into config.js and add this code:

var config = {
  KEY : 'YOUR TICKETMASTER API KEY'
}

3. Add it to your git ignore file as /app/assets/javascripts/config.js to protect your key
4. Bundle install
5. rails db:migrate
